# **mygit, a Git implementation in rust**
## **usage**
Set the git config in `main.rs`.<br/>
<br/>
Build the `mygit` binary and add it to your PATH:<br/>
<br/>
`$ cargo build`<br/><br/>
Add it to your PATH.<br/><br/>
Finally,initialize a Git repo and create commit:<br/><br/>
`$ mygit init`<br/>
`$ mygit add`<br/>
`$ mygit rm`<br/>
`$ mygit commit`<br/>
`$ mygit branch`<br/>
`$ mygit checkout`<br/>
